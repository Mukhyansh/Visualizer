# Visualizer

An interactive web-based visualizer for sorting and searching algorithms built using HTML, CSS, and JavaScript. It helps users understand algorithm behavior through real-time animations.

# Features

~Sorting Algorithms:
Bubble, Selection, Insertion
Merge, Quick, Heap

~Searching Algorithms
Linear Search
Binary Search

*Adjustable array size & animation speed*
*Randomize & reset array*
*Visual step-by-step comparisons and swaps*
*Algorithm search filter*

# Tech Stack

HTML5
CSS3
JavaScript
